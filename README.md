Projet : Base de Données Cinéma
Ce dépôt contient un script SQL complet pour créer et gérer un catalogue de films. Il a été réalisé dans le cadre de ma deuxième année de licence en informatique.
Ce que fait ce projet
Le script organise les informations autour de trois axes principaux :
• Les Films : Titres et dates de sortie.
• Les Acteurs : Identité des artistes.
• Les Catégories : Classification par genre (SF, Western, Drame, Comédie).
Contenu technique
Le fichier SQL inclut :
1.	La Structure : Création des tables avec gestion des clés étrangères pour lier les films aux acteurs et aux catégories.
2.	Le Jeu de Données : Insertion de films célèbres et d'acteurs connus pour tester la base.
3.	Les Requêtes : Trois exemples de recherches (recherche croisée d'acteurs, filtrage par genre et statistiques par catégorie).
Utilisation
Il suffit d'exécuter le script dans un environnement SQL pour générer la base de données et tester les requêtes.
